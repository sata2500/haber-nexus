# HaberNexus Dokümantasyon

Bu klasör, HaberNexus projesinin tüm dokümantasyonunu içerir. Dokümantasyon, kolay erişim için kategorilere ayrılmıştır.

## 📁 Klasör Yapısı

### 📊 Reports (Raporlar)

Proje geliştirme sürecinde oluşturulan tüm raporlar.

- **Geliştirme Raporları:** Özellik geliştirme ve iyileştirme raporları
- **Hata Düzeltme Raporları:** Bug fix ve sorun giderme raporları
- **Performans Raporları:** Optimizasyon ve performans iyileştirme raporları
- **Analiz Raporları:** AI, dashboard ve özellik analizleri

[Reports klasörüne git →](./reports/)

### 📖 Guides (Kılavuzlar)

Geliştirici kılavuzları, teknik dokümantasyon ve nasıl yapılır rehberleri.

- **Developer Guide:** Geliştirme süreçleri ve en iyi pratikler
- **AI Features Guide:** AI özellikleri kullanım kılavuzu
- **Technical Guides:** ESLint, GitHub Actions, Cron ve diğer teknik konular

[Guides klasörüne git →](./guides/)

### 📋 Plans (Planlar)

Özellik geliştirme planları ve yol haritaları.

- **Feature Plans:** Yeni özellik geliştirme planları
- **Development Roadmap:** Proje yol haritası
- **Architecture Plans:** Mimari tasarım planları

[Plans klasörüne git →](./plans/)

### 🚀 Deployment (Dağıtım)

Dağıtım talimatları ve konfigürasyonlar.

- **Deployment Instructions:** Adım adım dağıtım kılavuzu
- **Vercel Deployment:** Vercel platformu için özel talimatlar
- **Environment Setup:** Ortam değişkenleri ve konfigürasyon

[Deployment klasörüne git →](./deployment/)

### 🔍 Analysis (Analizler)

Proje analiz dokümanları ve araştırma notları.

- **Project Analysis:** Proje yapısı ve mimari analizleri
- **Problem Analysis:** Sorun tespiti ve çözüm analizleri
- **Feature Analysis:** Özellik analiz dokümanları

[Analysis klasörüne git →](./analysis/)
